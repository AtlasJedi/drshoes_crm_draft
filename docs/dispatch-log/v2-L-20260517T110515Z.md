## Motivation

Playwright audit (1440×900) found three routes with page-level overflow:
- `/admin/orders` — 239px: filters/tabs/presets scrolled with the list
- `/admin/clients` — 81px: search box scrolled with the table
- `/admin/sklep` — 127px: filter chips scrolled with the product grid

Owner directive: "strona nigdy nie jest scrollable — filtry przypięte na górze, pole z listą scrollowalne." Approach A locked: layout `<main>` keeps `p-6` padding, each page restructures internally.

## Pattern Applied

```tsx
<div className="h-full flex flex-col">
  <div className="shrink-0 ...">  {/* sticky chrome */}
  <div className="flex-1 min-h-0 overflow-auto">  {/* scrollable region */}
</div>
```

`min-h-0` on every flex-col parent is critical — without it flex children don't know their height and `overflow-auto` has no effect.
