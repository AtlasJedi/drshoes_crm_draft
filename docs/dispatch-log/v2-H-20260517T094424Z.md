## What It Does

Paint-fill animation on the status-change buttons in the OrderDrawer:
- Hover triggers a fill that slides up from below (CSS `::before` translateY)
- A skewed white stripe sweeps across on hover (CSS `::after` skewX + translateX)
- Active button stays filled (`.active` class) with non-interactive pointer cues
- Per-status color binding via CSS custom property `--c` and `.s-*` modifier classes
