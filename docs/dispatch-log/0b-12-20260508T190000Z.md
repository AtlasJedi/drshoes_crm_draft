## Decisions / deviations from plan

1. **`middleware.ts` → `proxy.ts`**: Next.js 16 deprecated `middleware.ts` in favour of `proxy.ts`, and requires the export function to be named `proxy` (not `middleware`). Initial build with `middleware.ts` produced a deprecation warning; renamed + re-exported as `proxy`. Build clean with no warnings after fix.

2. **`AdminSidebar` extracted**: `layout.tsx` would have exceeded 80 LOC with full sidebar JSX inline. Extracted to `apps/web/components/admin/AdminSidebar.tsx` as a server component per the granularity constraint.

3. **`next lint` skipped**: per task brief — known broken in Next.js 16. Noted here.

4. **`i` key removed from `.map()` callback**: plan used `(label, i) => ... key={i}` which triggers react/no-array-index-key; replaced with `key={label}` (labels are unique) — cleaner and TS strict-safe.
